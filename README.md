# proteosim-
Short workflow for proteomics
Proteosim Course Package
Proteosim ist ein kleines Python-Package zur Simulation grundlegender Schritte der Bottom-Up-Proteomics.
Es enthält Funktionen zum:

Einlesen von Proteinsequenzen aus FASTA-Dateien

Verdauen von Proteinen mit verschiedenen Proteasen

Berechnen der Sequenz-Coverage

Simulieren von LC-Retention Times (auf Basis des Pyteomics Achrom-Modells)

Validieren des Codes über automatisierte Unit Tests und GitHub Actions
